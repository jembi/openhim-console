Certificates

Manage the OpenHIM keystore. A new OpenHIM server certificate can be added or generated and trusted client certificates can also be added or edited. Only self signed client certificates that should be trusted need to be added to the trust store. The OpenHIM already trusts all certificates signed by the common CAs, although, certificates must be linked to a client on the clients page before they can be used to send in requests.
